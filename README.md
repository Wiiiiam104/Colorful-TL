# Colorful-TL
### GreasyFork版は[こちら](https://greasyfork.org/ja/scripts/441551-colorful-tl)<br>

AtCoderのRatingに基づいてTwitterのTimeLineを着色するUserScriptです.<br>
AtCoderのRatingに基づく色は2週間に1度自動で更新されます.<br><br>

※TwitterAPIにより@AtCoder_tl氏のリストを取得していますが、このUserScriptとは一切関係ありません<br>
※挙動がおかしい場合はCookieを削除すると直ることがあります.<br><br>

This is a UserScript code which color Twitter's TimeLine based on AtCoder rating.
